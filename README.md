# narajangteo-crawl
알바 부탁을 받았던, 간단히 만든 조달청 - 나라장터 크롤러 

원하는 키워드를 ``keyword.txt`` 에 개행으로 구분지어 적어두고,

``date.txt``에 두줄로 YYYY/MM/DD 형식으로 시작, 종료날짜를 적은 뒤에

``node index.js`` 를 통해 실행시키면 된다.
